# MazeRunner

**MazeRunner** — это консольный проект на Kotlin для генерации и решения лабиринтов.  
Проект демонстрирует работу с алгоритмом генерации лабиринта, поиском пути (DFS) и базовой файловой системой для сохранения/загрузки лабиринтов.

---

## Функционал

- Генерация нового лабиринта произвольного размера
- Загрузка лабиринта из файла
- Сохранение лабиринта в файл
- Отображение лабиринта в консоли
- Поиск пути от старта до выхода с визуализацией
